使用docusaurus2创建博客并部署至github pages

May 29, 2020 · 1 min read

vacantthinker

1前置需求#

  • node.js (版本 >= 10.15.1)

  • yarn (版本 >= 1.5)

2创建docusaurus v2项目#

your-website: 你的网站文件夹

npx @docusaurus/init@next init your-website classic
npx @docusaurus/init@next init vacantthinker.github.io classic

3项目结构#

your-website
├── blog
│ ├── 2019-05-28-hola.md
│ ├── 2019-05-29-hello-world.md
│ └── 2020-05-30-welcome.md
├── docs
│ ├── doc1.md
│ ├── doc2.md
│ ├── doc3.md
│ └── mdx.md
├── src
│ ├── css
│ │ └── custom.css
│ └── pages
│ ├── styles.module.css
│ └── index.js
├── static
│ └── img
├── docusaurus.config.js
├── package.json
├── README.md
├── sidebars.js
└── yarn.lock

4本地测试运行#

cd my-website
npm run start
cd my-website
yarn run start

5配置docusaurus.config.js文件#
